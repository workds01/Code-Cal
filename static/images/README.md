# Calendar Images

This directory should contain the following images for the calendar:

1. `tiger-mascot.png` - The main tiger mascot displayed at the top of the calendar
2. Monthly tiger illustrations:
   - `january.png` - Skiing tiger
   - `february.png` - Valentine tiger
   - `march.png` - Spring tiger with umbrella
   - `april.png` - Tiger with flowers
   - `may.png` - Tiger flying kite
   - `june.png` - Swimming tiger
   - `july.png` - Beach tiger
   - `august.png` - Summer tiger
   - `september.png` - School tiger
   - `october.png` - Halloween tiger
   - `november.png` - Autumn tiger
   - `december.png` - Winter holiday tiger

## Image Requirements
- All images should be PNG format with transparent backgrounds
- Recommended size: 150px width, maintaining aspect ratio
- Use a consistent art style across all illustrations
- Keep file sizes optimized (< 100KB per image)

## Placeholder Images
Until you have the custom illustrations, you can:
1. Use a placeholder tiger mascot image
2. Use solid color blocks with the month name for monthly illustrations
